# Эхолингв

Платформа для изучения балкарского языка с интерактивными уроками, личным словарным архивом и отслеживанием прогресса.

## Особенности

- 📚 **5 полноценных уроков** с теорией и практическими упражнениями
- 📖 **Личный архив слов** с возможностью добавления, редактирования и экспорта
- 📊 **Отслеживание прогресса** и статистика в личном кабинете
- 🐱 **Милый талисман** в виде серого котика
- 🎨 **Современный дизайн** в стиле Duolingo

## Установка

1. Убедитесь, что у вас установлен Node.js (версия 16 или выше)
2. Клонируйте или скачайте проект
3. Установите зависимости:

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

Собранное приложение будет находиться в директории `dist`

## Добавление новых уроков

Новые уроки добавляются в файл `src/data/lessons.js`. Каждый урок имеет следующую структуру:

```javascript
{
  id: 'lesson-6',
  order: 6,
  title: 'Название урока',
  description: 'Описание урока',
  sections: [
    {
      type: 'theory',
      title: 'Теоретический блок',
      content: [
        { balkar: 'слово', russian: 'перевод' }
      ]
    },
    {
      type: 'exercise',
      exerciseType: 'translate',
      title: 'Упражнение',
      questions: [
        { question: 'Вопрос', answer: 'Ответ' }
      ]
    }
  ]
}
```

## Возможности приложения

### Уроки
- Теоретические блоки с лексикой
- Интерактивные упражнения (перевод, выбор варианта)
- Проверка ответов с обратной связью
- Последовательное открытие уроков

### Архив слов
- Добавление новых слов
- Редактирование и удаление
- Отметка слов как "изученные"
- Поиск и фильтрация
- Экспорт в JSON и CSV

### Личный кабинет
- Статистика прогресса
- Количество пройденных уроков
- Количество изученных слов
- Сброс прогресса
